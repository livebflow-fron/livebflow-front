# 🏦 LivebFlow - Sistema de Gestão Interna

## 📖 Sobre o Projeto
O **LivebFlow** é o sistema de gestão interna de parceiros e acompanhamento de propostas de fiança locatícia, desenvolvido para a empresa LivebBank. Este repositório contém a interface de usuário (Front-end) estática do sistema, incluindo telas de Login, Kanban de propostas e Cadastro de Imobiliárias.

Projeto desenvolvido para a disciplina de Projeto Integrador III-B do curso de Análise e Desenvolvimento de Sistemas.

## 🚀 Tecnologias Utilizadas
* **Front-end:** HTML5, CSS3, Tailwind CSS
* **Back-end (Em breve):** Java, Spring Boot
* **Banco de Dados (Em breve):** PostgreSQL
